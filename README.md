## 📂 Project Showcase

### 🧠 Customer Churn Prediction

Predicts telecom customer churn risk and returns probability-driven outputs for retention workflows.

- Multi-model classification with tuning and evaluation
- FastAPI serving for real-time churn inference
- Automated portfolio assets and real prediction samples

Tech: Python, scikit-learn, XGBoost, FastAPI, Matplotlib, Seaborn  
👉 [View Project](./customer_churn_prediction)

### 🛡️ Fraud Detection System

Detects suspicious financial transactions with threshold-aware decisioning and imbalance handling.

- Imbalance strategies (none, oversampling, SMOTE)
- Precision-recall threshold tuning and rich evaluation artifacts
- API scoring with strict schema checks and production logging

Tech: Python, scikit-learn, imbalanced-learn, XGBoost, FastAPI  
👉 [View Project](./fraud_detection_system)

### 🎬 Recommendation Engine

Delivers personalized recommendations using collaborative, content-based, and hybrid ranking signals.

- Hybrid recommender with strategy comparison
- Cold-start fallback and recommendation caching
- API-ready top-N recommendation serving

Tech: Python, scikit-learn, Pandas, FastAPI, recommendation modeling  
👉 [View Project](./recommendation_engine)

## ⚙️ How To Use These Projects

1. Clone repository

```bash
git clone <your-repo-url>
cd ML_Projects
```

1. Create and activate environment

```bash
python -m venv .venv
.venv\Scripts\activate
```

1. Install dependencies per project

```bash
cd customer_churn_prediction
pip install -r requirements.txt
```

1. Run training or showcase pipelines

```bash
python -m src.pipelines.training_pipeline
python -m src.pipelines.portfolio_showcase_pipeline
```

1. Serve API

```bash
uvicorn api.main:app --host 0.0.0.0 --port 8000 --reload
```
